# Mini SaaS Template Store

Full Stack Web Developer Intern Task

## Tech Stack

Frontend:
- React.js (Vite)
- TailwindCSS
- Axios
- React Router

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt

---

## Setup Instructions

### Backend

cd server

npm install

Create .env file:

PORT=5000
MONGO_URI=mongodb://127.0.0.1:27017/template_store
JWT_SECRET=super_secret_change_me

Run backend:

npm run dev

---

### Frontend

cd client

npm install

npm run dev

Open browser:
http://localhost:5173

---

## Features

- User Registration & Login
- JWT Authentication
- View Templates
- Favorite Templates
- View Favorite Templates
- Protected Routes
- Search & Filter Templates
